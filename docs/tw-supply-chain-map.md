# 台股產業上下游供應鏈地圖（新手版）

寫給股市新手的台股產業結構入門。搭配 `docs/cluster-classification-guide.md` 使用：
那份講「怎麼分族群」，本檔講「台股有哪些供應鏈、每條鏈的上中下游是誰」。

> 個股僅列各環節的代表股（非完整名單、非推薦買進）。公司業務會變，重大決策前請再確認。

---

## 零、先懂三個詞：上游、中游、下游

想像一條河：**上游**做原料、**中游**做加工製造、**下游**做成品與銷售。

- **上游**：原物料、關鍵零件、設計圖（例：矽晶圓、銅箔、IC 設計）
- **中游**：把原料變半成品（例：晶圓代工、面板製造、PCB 板廠）
- **下游**：組裝成品、賣給消費者（例：組裝代工、品牌、通路）

**為什麼重要**：一條供應鏈是「一榮俱榮、一損俱損」。下游接到大單，中上游遲早跟著旺；
上游原料漲價，中下游成本就受壓。族群行情（雁行擴散）就是沿著供應鏈擴散的——
龍頭先漲，資金再沿上下游找「還沒漲的同鏈股票」。

## 一之一、記憶體（半導體家族中的獨立鏈）

記憶體與晶圓代工**共用最上游**（矽晶圓、設備），但它有自己完整的上下游、
自己的行情引擎——**DRAM／NAND 報價循環**：報價漲，全鏈同漲；報價跌，全鏈同冷，
節奏與台積電代工鏈經常脫鉤，所以市場把它當一條獨立的鏈（獨立族群）操作。

| 環節 | 做什麼 | 代表股 |
|---|---|---|
| 利基型記憶體 IC 設計（上游） | 設計特殊規格 DRAM，交代工廠生產 | 鈺創 5351、晶豪科 3006、愛普* 6531 |
| 標準型 DRAM 製造（中游核心） | 大宗記憶體，直接跟著國際報價走 | 南亞科 2408 |
| 利基型製造 | 少量多樣，車用／工控用 | 華邦電 2344、旺宏 2337（NOR Flash） |
| 控制 IC | SSD／記憶卡的「大腦」 | 群聯 8299、點序 6485 |
| 記憶體封測（下游） | 專攻記憶體的封裝測試 | 力成 6239、南茂 8150、華東 8110 |
| 模組（下游） | 把顆粒組裝成記憶條／SSD | 威剛 3260、創見 2451、十銓 4967、宇瞻 8271 |
| 通路（最下游） | 記憶體大盤商 | 至上 8112（三星系） |

**與其他鏈的連動**：
- **往上游**：記憶體漲價 → 擴產 → 矽晶圓（環球晶、合晶、中美晶）與設備跟著受惠
  （2026-07 的族群突破就出現過「記憶體漲價向上游延伸至矽晶圓」的行情）
- **往下游**：HBM（高頻寬記憶體）是 AI 伺服器的關鍵零件，讓記憶體鏈與 AI 鏈掛勾
- **看行情先看報價**：DRAM／NAND 合約價與現貨價是這條鏈的領先指標，報價翻揚常領先股價啟動

## 一之二、台積電擴產／蓋廠與廠務（資本支出概念股）

台積電在台灣、美國、日本、德國同步擴產，一座先進製程廠造價數千億，
錢會依序流向「蓋廠 → 裝機 → 營運耗材」三個階段的供應商。
這條鏈的行情引擎是**台積電資本支出（CapEx）**：法說會宣布上調資本支出，整條鏈跟著動。

| 階段／環節 | 做什麼 | 代表股 |
|---|---|---|
| 土建營造（最前段） | 蓋廠房結構體，預鑄工法可縮短工期 | 潤弘 2597、達欣工 2535、根基 2546 |
| 統包工程 | 整廠設計＋監造的總承包 | 中鼎 9933 |
| 無塵室工程 | 晶圓廠心臟：潔淨室統包 | 漢唐 2404、聖暉* 5536、亞翔 6139 |
| 機電工程 | 電力、空調、消防系統 | 洋基工程 6691 |
| 廠務系統／管路 | 特殊氣體與化學品的輸送管線 | 帆宣 6196、朋億* 6613、信紘科 6667 |
| 電力設備 | 超高壓變電、配電（與重電鏈重疊） | 中興電 1513、華城 1519 |
| 無塵室耗材（營運期） | 濾網、耗材的長期換裝需求 | 濾能 6823 |
| 設備零組件／清洗（營運期） | 設備零件在地化、機台零件清洗再生 | 京鼎 3413、翔名 8091、世禾 3551 |

**新手看點**：
- **時間差**：宣布建廠 → 土建先動（1~2 年）→ 無塵室與機電（後 1 年）→ 裝機與耗材（量產後持續）。
  資金會沿這個順序輪動，土建股先反應、耗材股最晚但吃得最久
- **海外廠加分**：美日德廠的工程單價更高，拿到海外訂單的廠務股（漢唐、帆宣、亞翔）常被市場另眼看待

## 一之三、先進製程材料／特用化學品（半導體上游的上游）

晶圓廠 24 小時運轉要持續「餵」材料：光阻、特殊氣體、研磨液、電子級溶劑。
這條鏈的特性是**耗材型生意**——不管台積電賺不賺錢，開機就要用，營收穩定；
行情引擎是**先進製程推進**（越先進用量越大、規格越高）與**國產化替代**（地緣政治下把日本貨換成台灣貨）。

|---|---|---|
| 光阻／顯影材料 | 微影製程的感光材料（最難國產化） | 永光 1711、達興材料 5234 |
| 材料代理／通路 | 代理日本信越等大廠的光阻、石英、矽晶圓 | 崇越 5434 |
| 特殊氣體／前驅物 | 沉積、蝕刻製程用的高純度氣體 | 台特化 4772、晶呈科技 4768 |
| 電子級溶劑／化學品 | 清洗、剝離用的高純度溶劑 | 勝一 1773、三福化 4755 |

| CMP 研磨耗材 | 晶圓平坦化的鑽石碟、研磨液 | 中砂 1560 |

| 靶材／貴金屬回收 | 濺鍍靶材與貴金屬循環 | 光洋科 1785 |
| 光罩 | 電路圖的「底片」 | 台灣光罩 2338 |
| EUV 周邊 | EUV 光罩傳送盒（Pod） | 家登 3680 |

**新手看點**：
- **與蓋廠鏈的關係**：蓋廠鏈吃「一次性工程財」，材料鏈吃「長期耗材財」——廠蓋越多，材料的長期需求越大，兩條鏈是同一個資本支出故事的前後段
- **國產化題材**：日本管制或供應鏈風險新聞出現時，這族群常整體點火（把進口材料換成台廠）
- **先進製程受惠排序**：製程越先進，微影次數越多 → 光阻、EUV 周邊（家登）彈性最大

## 附：新手怎麼用這張地圖

1. **看到一檔股票漲，先問它在哪條鏈的哪個環節** → 同環節的同伴（族群）是誰？上下游是誰？
2. **族群行情的擴散順序**通常是：龍頭股先動 → 同環節同伴跟上 → 再沿上下游蔓延
   （對應 `cluster-classification-guide.md` 的雁行擴散與候補族群框架）
3. **環節 ≠ 產業別**：交易所的「電子零組件業」裡同時有被動元件、PCB、連接器三條不同的鏈，
   分析時要拆到環節層級
4. 一檔股票可以同時屬於多條鏈（台達電既是 AI 伺服器電源、也是電動車充電樁）——
   看當下市場在炒哪個題材，它就跟哪個族群連動

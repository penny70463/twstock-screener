🎯 族群轉強判斷框架

若以下所有跡像同時出現，族群轉強確定性極高：

✓
高成交量訊號
同族群 2~3 檔個股成交量達到 2x 20日均量以上，這是「點火」的必備條件
✓
族群集聚度（30%+）
該族群 30% 以上的股票在 1~3 天內同時點火，呈現「雁行擴散」，確認族群啟動而非個別炒作
✓
波浪時序確認
龍頭股或旗手股已在 3~7 天前觸發，形成「波浪啟動序列」，後進股開始異動
✓
焦點日期前 1~2 天的異常
在族群集體點火前 1~2 天，候補股開始「小量異常」或「技術突破」（如 MA 排列）
✓
題材驅動明確
有明確的外部驅動（產業新聞、政策、地緣政治等），而非純技術面自發

⚠️ 進場時機建議

  • 黃金進場：龍頭股觸發 → 候補股異常 → 族群集體點火前 1 天
    此時漲幅空間最大（8~15%），但需要確認族群啟動訊號
  • 安全進場：族群集體點火當日 → 前高 (high) 位置
    此時風險最低，但漲幅空間已部分消化，適合保守型投資
  • 高風險進場：成交量 6x+ 且族群觸發率 > 75%
    此時已在相對高位，可視為波段中期位置，需要設置嚴格停損
